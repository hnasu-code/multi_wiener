The repository contains the following files.

(1) func_M0.R
- The R function that performs EM algorithm for model M0

(2) func_simu.R
- The R function that generates simulated degradation data for various models

(3) func_start.R 
- The R function that finds the helpful initial starting point for the EM algorithms.

(4) simu_M0.R
- The R codes that conduct simulation study for model M0
