The repository contains the following files:

(1) func_simu.R
- The R function that generates simulated degradation data.

(2) run_figure_2.R
- The R codes that generate Figure 2.

(3) func_start.R 
- The R function that finds the helpful initial starting point for the EM algorithm.

(4) func_M0.R
- The R function that performs EM algorithm for model M0.

(5) simu_M0.R
- The R codes that conduct simulation study for model M0.
